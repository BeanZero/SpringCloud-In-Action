##SpringCloud In Action

**SpringCloud微服务集成：适用于微服务的学习或者二次开发**



### 模块说明

```
├─common-dependencies ------------- 统一父依赖：公用jar、版本号等信息。一般存放在Maven私服仓库中。
├─cloud-config -------------------- 分布式配置中心
├─cloud-eureka -------------------- 服务注册中心(注册与发现)
├─cloud-zipkin -------------------- 分布式链路跟踪(待续)
├─cloud-zuul   -------------------- 路由网关(待续)
├─cloud-admin  -------------------- 服务监控(待续)
├─server-xxxx  --------------------(待续)
├─xxxxxxxxxxx  --------------------(待续)
```
